# TextCard

将长文转换为精美长图的在线工具。

## 功能特性

- 📝 输入长文内容，支持多段落自动排版
- 🎨 8 种精美主题：极简白、暗夜绿、暖阳、深海、日落、森林、薰衣草、报纸
- 📏 可调节字体大小 (14-28px)
- 👁 实时预览
- 📥 一键导出 PNG 长图

## 使用方法

1. 在左侧文本框输入或粘贴长文内容
2. 可选填写标题和作者
3. 选择喜欢的主题风格
4. 调整字体大小
5. 点击「生成长图」按钮下载

## 技术栈

- HTML5
- Tailwind CSS
- html2canvas
- Google Fonts (Space Grotesk, Noto Serif SC)

## 运行

直接用浏览器打开 `index.html` 即可使用。
